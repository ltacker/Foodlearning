# Foodlearning 🥔🍋🌮🥕
Application layer of Foodlearning project


## Prerequisites ##
1. Java >= 1.8.0_144
2. NodeJs >= 6.11.3
3. sbt >= 1.0.2

## Build & Run ##

```sh
$ cd [working directory]
$ ./sbt
> run
```
Then open [http://localhost:8080/](http://localhost:8080/) in your browser.
